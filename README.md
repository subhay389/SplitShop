# SplitShop
 An App for online shopping and splitting money with friends.

* Ashok Tamang

## Using This Project
* `git clone https://github.com/subhay389/SplitShop.git`
* `ionic state restore`
* `ionic platform add android` 
* `ionic run android` 
* `ionic platform add ios` 
* `ionic run ios`
